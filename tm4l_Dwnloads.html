Downloads

This software is based on TMAPI, and shipped with TM4J, a Java open source library implementing a topic maps engine.

The TM4L distribution files include everything you need to run the Editor or the Viewer except the Java Runtime Environment (Java JRE or SDK, version 1.5 ). For TM4L 1.4 there is also a version available using Java 1.4.

Installation and Help

Unzip the package and run it by double clicking run.bat (on Windows) or run.sh (on Unix). For more information about using the TM4L Editor and Viewer please refer to the Resource page on this site, which contains user guides. After unzipping the packages, in the created doc directory you will also find the user guide for the Editor/Viewer.
